# Pop ups

Create a character vector that summarizes and formats all variables in a
data frame.

## Usage

``` r
s_popup(x, r, my_variables, pretty_names, digits = 0)
```

## Arguments

- x:

  parameter name and vector type.

- r:

  parameter name and vector type.

- my_variables:

  parameter name and vector type.

- pretty_names:

  parameter name and vector type.

- digits:

  (default 0).

## Value

Vector kind output by the function and short description if is
necessary.
